# News API Project 

This is a simple Python project that fetches the latest news articles using the **NewsAPI and displays their titles and URLs.

The project is useful for beginners to understand:
- API integration
- JSON data handling
- HTTP requests in Python

---

# Features
- Fetches latest news based on a search query
- Displays news title and article link
- Uses `requests` library
- Simple and easy-to-understand code

---

# Technologies Used
- Python 3
- NewsAPI
- requests library
